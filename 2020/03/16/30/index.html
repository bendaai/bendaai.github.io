30. 串联所有单词的子串

典型的滑动窗口解字符串题

题目内容

给定一个字符串 s 和一些长度相同的单词 words。找出 s 中恰好可以由 words 中所有单词串联形成的子串的起始位置。

注意子串要与 words 中的单词完全匹配，中间不能有其他字符，但不需要考虑 words 中单词串联的顺序。

示例

示例 1：

输入：
s = “barfoothefoobarman”,
words = [“foo”,”bar”]
输出：[0,9]
解释：
从索引 0 和 9 开始的子串分别是 “barfoo” 和 “foobar” 。
输出的顺序不重要, [9,0] 也是有效答案。
示例 2：

输入：
s = “wordgoodgoodgoodbestword”,
words = [“word”,”good”,”best”,”word”]
输出：[]

题解

class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
        '''
        例子：比如示例1
        将s的可能性分为k组，k为words中字符串的长度
        1.barfoo foothe thefoo foobar barman
        2.arfoot oothef hefoob oobarm
        3.rfooth othefo efooba obarma
        每一组中，后一个字符串是前一个字符串减了头k个字符加了尾k个字符得到
        我们用t维护前一个字符串，按长度k切分后的情况，比如barfoo t表示为{'bar':1,'foo':1}
        若t和words得到的vc一样，那么形成匹配。
        '''
        if not s or not words:
            return []
        ans=[]
        from collections import Counter
        vc=Counter(words)
        k=len(words[0])
        L=len(words)
        
        for i in range(k):
            t=Counter()
            cnt=0
            l,r=i,i
            while r+k<=len(s):
                w=s[r:r+k]
                r+=k
                if w not in vc:
                    l=r
                    t.clear()
                    cnt=0
                else:
                    cnt+=1
                    t[w]+=1
                    while t[w]>vc[w]:
                        l_w=s[l:l+k]
                        l+=k
                        cnt-=1
                        t[l_w]-=1
                    if cnt==L:
                        ans.append(l)
        return ans
